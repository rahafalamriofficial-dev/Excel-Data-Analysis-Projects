# Rahaf's Excel Dashboards Portfolio

Welcome to my personal Excel portfolio.  
This repository showcases a collection of dashboards and data analysis projects I created independently to enhance my analytical and visualization skills.

---

## Projects Included

### 1. Salary Dashboard
**Goal:** Analyze and visualize salary data to understand compensation trends.  

**Key Insights:**
- Average salary by department and job title  
- Gender pay comparison  
- Highest and lowest salaries  
- Interactive filters for departments and positions  

**Tools Used:** Excel (Pivot Tables, Pivot Charts, Conditional Formatting, Slicers)

---

### 2. Project Analysis Dashboard
**Goal:** Evaluate the performance and progress of multiple projects to identify trends and areas of improvement.  

**Key Insights:**
- Project completion status (Completed, Ongoing, Delayed)  
- Comparison between planned and actual budgets  
- Percentage of completion and overall efficiency  
- Conditional formatting to highlight project performance  

**Tools Used:** Excel (Pivot Tables, Charts, Conditional Formatting, Formulas like IF & COUNTIFS)
